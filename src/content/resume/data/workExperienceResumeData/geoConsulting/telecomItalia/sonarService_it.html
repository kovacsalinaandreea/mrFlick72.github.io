The goal of project is been create a SOAP/REST service interface for download the src from Serena Software PVCS and Apache SVN. The interface permitted also the start of a Code quality inspection through SONARQube. This service are consumed by a SharePoint 2010 WebPart, that, after permitted at the end-user of selected the src in base at the Telecom Italia criteria, start a service that in pipeline start the download of src and start the inspection.
